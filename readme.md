## _Installation_

Clone and install the dependencies and devDependencies and start the server.

```sh
# Clone the repository
cd Front
# install dependencies
$ npm install
# serve with hot reload at localhost:3000
$ npm run dev

# Clone the repository
cd Back
# install dependencies
$ npm install
# serve with hot reload at localhost:3000
$ npm run start
```
